# What's New in RVU Counter v3.1.9

## Customizable Display Order

- Drag-and-drop reordering of main stats in Settings > Display
- Per-row toggles for both counter and compensation visibility
- Reset to default order with one click

## Monthly Projection

- New "Est. this month" stat showing projected monthly RVU and income
- Click the value to configure target shifts and extra hours
- Settings auto-reset each month

## Bug Fixes

- Fixed Dropbox backup listing failing after first backup (duplicate Authorization header)
- Fixed Dropbox enable checkbox not actually enabling backups (DropboxEnabled vs DropboxBackupEnabled mismatch)
